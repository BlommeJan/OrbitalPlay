The OrbitalPlay Controller

The OrbitalPlay controller combines a sturdy wooden frame with 3D-printed components to enhance its aesthetic and functionality. It features six omnidirectional ball bearings that allow the central ball to move effortlessly in all directions. A sensor beneath the ball tracks its movement, ensuring precise input. Powered by a Teensy microcontroller, the controller transmits data to your PC, making it easy to use with OrbitalPlay games.

Recreate It Yourself

Everything you need to build the OrbitalPlay controller is available on the GitHub repository :

  • Bill of Materials (BOM): A complete list of components.
  • 3D Print Files: Ready-to-use designs for the hardware.
  • Step-by-Step Instructions: Detailed guides to assemble and configure the controller.

With these resources, you can recreate the hardware and fully immerse yourself in the OrbitalPlay experience.
